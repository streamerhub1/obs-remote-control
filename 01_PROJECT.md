# О проекте

## Назначение

Streamly позволяет стримеру безопасно предоставить выбранным людям удалённый доступ к своему OBS.

Стример и модератор устанавливают одно приложение и входят через Twitch.

## Основной сценарий

1. Пользователь устанавливает приложение.
2. Авторизуется через Twitch.
3. Получает профиль и индивидуальный код, например `PH-8K3M-92QX`.
4. Стример подключает приложение к локальному OBS.
5. Нажимает «Добавить модератора».
6. Вводит индивидуальный код другого пользователя.
7. Модератор принимает приглашение.
8. Стример задаёт разрешения.
9. Модератор открывает OBS стримера.
10. Приложения устанавливают P2P-соединение.
11. Модератор получает состояние OBS и управляет разрешёнными функциями.
12. Приложение стримера выполняет команды через локальный `obs-websocket`.

## Роли

Каждый аккаунт универсален.

Пользователь может:

- подключить свой OBS;
- добавить собственных модераторов;
- быть модератором нескольких стримеров;
- использовать несколько устройств.

## Сайт

Сайт содержит:

- главную;
- возможности;
- тарифы;
- скачивание установщика;
- Twitch OAuth;
- кабинет;
- подписку;
- устройства и сессии;
- privacy и terms.

Сайт не является панелью управления OBS и не проксирует постоянный OBS-трафик.

## Ограничения

- OBS и desktop-приложение стримера должны быть запущены.
- Полнота управления ограничена `obs-websocket` и API плагинов.
- Часть сетей потребует TURN.
- Stream key и пароль OBS не передаются модератору.
